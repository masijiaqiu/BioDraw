# "parameter-key"=default-value[optional-choice-if-available/data-type](description)
# All the parameters setting shold be string or float(int).
"""
This is a graph showing the cumulative line and the reverse cumulative line on given data. Looks better with normal distributed data.
"""
"title"=[string] (Set a title of the current axes.)
"xlabel"=[string] (Set the x axis label of the current axis.)
"ylabel"=[string] (Set the y axis label of the current axis.)
“num_bin”=[positive integer] (Set the number of the bins to a certain integer.)
“normed”=[“True”|”False”] (Set true to set the total area of bins to 1. In this diagram, to achieve the best viewing result for the Gauss curve, this parameter must be “True” or enter a positive number.)
"alpha"=[float] (Float 0.0 transparent through 1.0 opaque.)
“color1”=[string] (Setting the color for the cumulative step, this string must be written as “color”, for example: “red”. Optional.)
“color2”=[string] (Setting the color for the reverse cumulative step, this string must be written as “color”, for example: “red”. Optional.)
“curve”=[”True”] (This parameter is used for generating the cumulated Gaussian line for this graph. Only when this is set to “True” can the line be generated. Optional.)
